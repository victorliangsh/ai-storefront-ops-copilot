# AI Workflow Prompts

This document provides reusable AI prompts for Shopify and direct-to-consumer storefront operators.

The prompts are designed to help small ecommerce teams move faster while keeping the workflow practical, structured, and easy to review.

## 1. Product Selection Review

Use this prompt when evaluating a new product idea before building a storefront.

```text
You are an ecommerce product selection assistant.

Review the following product idea for a Shopify or direct-to-consumer storefront.

Product name:
[PRODUCT NAME]

Target customer:
[TARGET CUSTOMER]

Main customer problem:
[CUSTOMER PROBLEM]

Supplier notes:
[SUPPLIER NOTES]

Estimated cost and price range:
[COST AND PRICE]

Evaluate the product using these criteria:
- Clear customer problem
- Strong visual demonstration
- Simple sourcing
- Healthy margin potential
- Low compliance risk
- Low return risk
- One-product store fit
- Short-form content potential

Return:
1. Product score from 1 to 40
2. Strengths
3. Risks
4. Missing information
5. Recommendation: test, research more, or reject
```

## 2. Shopify Product Page Copy

Use this prompt to draft a product page from structured product inputs.

```text
You are a Shopify product-page copy assistant.

Create conversion-focused product-page copy for the following product.

Product name:
[PRODUCT NAME]

Target customer:
[TARGET CUSTOMER]

Main problem:
[MAIN PROBLEM]

Main benefits:
[BENEFITS]

Price range:
[PRICE]

Store style:
[ONE-PRODUCT STORE / GENERAL STORE]

Generate:
1. Product title
2. Short product description
3. Five bullet-point benefits
4. FAQ section
5. Trust section near the purchase button
6. Shipping and guarantee copy
7. Three short-form ad angles

Keep the copy clear, practical, and suitable for a small ecommerce storefront.
```

## 3. Storefront Audit

Use this prompt before sending traffic to a product page.

```text
You are a storefront conversion audit assistant.

Review this product page information and identify what should be improved before sending traffic.

Product:
[PRODUCT]

Target customer:
[TARGET CUSTOMER]

Page sections:
[PAGE SECTIONS]

Offer:
[OFFER]

Traffic source:
[TRAFFIC SOURCE]

Audit these areas:
- First-screen clarity
- Product benefit clarity
- Trust signals
- Mobile buying experience
- FAQ completeness
- Shipping and return clarity
- Checkout readiness

Return:
1. Audit summary
2. Top five issues
3. Recommended fixes
4. Must-fix items before traffic
5. Nice-to-have improvements
```

## 4. Daily Storefront Operations Review

Use this prompt to create a short daily review for an ecommerce store.

```text
You are an ecommerce operations assistant.

Create a daily operations review for this storefront.

Store type:
[STORE TYPE]

Product:
[PRODUCT]

Traffic notes:
[TRAFFIC NOTES]

Sales and checkout notes:
[SALES NOTES]

Customer messages:
[CUSTOMER MESSAGES]

Known issues:
[ISSUES]

Return:
1. Daily summary
2. What worked
3. What needs attention
4. Customer questions to answer
5. Product-page improvements
6. Next-day action list

Keep the report concise and action-oriented.
```

## 5. Launch Checklist Generator

Use this prompt to turn a product idea into a launch checklist.

```text
You are a Shopify launch planning assistant.

Create a launch checklist for the following product.

Product:
[PRODUCT]

Target customer:
[TARGET CUSTOMER]

Supplier status:
[SUPPLIER STATUS]

Product assets available:
[ASSETS]

Payment and shipping status:
[PAYMENT AND SHIPPING]

Generate a checklist covering:
- Product research
- Supplier verification
- Product-page copy
- Images and demo assets
- Offer setup
- Mobile storefront review
- Checkout and payment readiness
- FAQ and support information
- First traffic test
- Daily review process
```

## Notes

These prompts are starting points. Operators should review all generated content before publishing it to a storefront.

Avoid using AI output to make unsupported product claims, copy another brand's assets, or promise unrealistic results.
