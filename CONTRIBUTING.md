# Contributing

Thank you for your interest in AI Storefront Ops Copilot.

This project is focused on practical AI-assisted workflows for Shopify and direct-to-consumer storefront operators.

## Good Contributions

Useful contributions include:

- Product selection templates
- Storefront launch checklists
- Product-page copy examples
- Storefront audit examples
- Daily operations report examples
- Local tool improvements
- Documentation fixes
- Workflow prompts for ecommerce operators

## Contribution Guidelines

Please keep contributions practical, clear, and reusable.

When adding examples, avoid copying brand assets, product images, reviews, or copyrighted text from real stores. Use original examples or clearly generic sample content.

When adding ecommerce workflows, avoid making unrealistic performance claims. The project should help operators test and improve storefronts without promising revenue, conversion rates, or investment outcomes.

## Local Prototype

The local prototype is located at:

```text
src/index.html
```

It should remain simple enough for non-technical operators to open in a browser without a server or API key.

## Pull Request Checklist

Before submitting a pull request, check that:

- The contribution is relevant to ecommerce storefront operations.
- The content is original or safe to share.
- The language is clear for small merchants and operators.
- Markdown files render correctly on GitHub.
- Code examples do not require private credentials.

## Project Direction

The project is intentionally focused on storefront operations, product-page assets, launch workflows, and daily ecommerce reviews.

Please avoid adding unrelated features that would make the project harder for small operators to understand or use.
